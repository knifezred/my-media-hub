# Frontend Rules

Stack:

- Vue3
- TypeScript
- Pinia
- pnpm (package manager)

---

# Component Design

Prefer:

Small components

Single responsibility

Reusable UI

---

# State Management

Pinia only.

Avoid global mutable state.

---

# API Access

Centralize API requests.

Do not call fetch directly from pages.

---

# Routing

Pages:

views/

Reusable components:

components/

---

# Type Safety

Avoid any.

Prefer explicit types.

---

# UI Philosophy

Fast

Minimal

Content-focused

---

# Discovery First

Users should immediately see discoverable content.

Search and recommendation are primary experiences.

Management is secondary.
