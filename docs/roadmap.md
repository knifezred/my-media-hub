# 产品路线图（Roadmap）

项目名称：My Media Hub

版本：1.0

状态：进行中

---

# 项目愿景

My Media Hub 是一个面向 NAS 的个人媒体发现中心。

项目目标不是管理文件。

项目目标是帮助用户持续从自己的媒体资源库中发现值得消费的新内容。

支持媒体类型：

* 图片
* 视频
* 小说

未来扩展：

* 漫画
* 音乐
* PDF
* 课程
* 网页存档

# 长期愿景

My Media Hub 将成为 NAS 中所有媒体资产和知识资产的统一发现层。

系统持续帮助用户发现：

* 被遗忘的内容
* 高价值内容
* 与兴趣相关的内容

而不依赖用户手动整理资源库。
